# Cheezies Gourmet

Self-hosted React + Vite website. Base44 dependencies removed.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
# output in /dist
```

## Deployment

Hosted on Vercel. Every push to `main` auto-deploys.

## Updating Events

Edit `src/data/events.json` to add or update events — no code changes needed.
